# Лабораторна робота №6: Віртуальні середовища в Python

## Мета роботи

Вивчити основні інструменти для управління віртуальними середовищами та залежностями в Python проектах, а саме:

- **venv** - вбудований інструмент Python для створення віртуальних середовищ
- **Pipenv** - сучасний інструмент для управління залежностями та середовищами
- **Poetry** - інструмент для управління залежностями та пакування Python проектів

## Структура проекту

```
6_lab/
├── anime.py              # Скрипт для роботи зі змінними оточення
├── app.py                # Flask веб-додаток з Jikan API
├── README.md             # Основна документація
├── 1_venv/               # Приклад роботи з venv
│   ├── requirements.txt
│   └── requirements-dev.txt
├── 2_pipenv/             # Приклад роботи з Pipenv
│   ├── Pipfile
│   └── run.sh
└── 3_poetry/             # Приклад роботи з Poetry
    ├── pyproject.toml
    ├── mkdocs.yml
    └── docs/
```

## Опис завдання

У рамках лабораторної роботи було створено простий веб-додаток на Flask, який використовує **Jikan API** для отримання інформації про аніме. Проект демонструє роботу з різними менеджерами віртуальних середовищ.

### Використані бібліотеки

- **Flask 3.1.3** - фреймворк для створення веб-додатків
- **jikanpy-v4 1.0.2** - Python обгортка для Jikan API (MyAnimeList)
- **flake8 7.3.0** - інструмент для перевірки якості коду (dev-залежність)
- **mkdocs 1.6.1** - генератор статичної документації (docs-залежність)

## Основні поняття

### Віртуальне середовище

Віртуальне середовище (virtual environment) - це ізольоване Python оточення, яке дозволяє:

- Встановлювати пакети для конкретного проекту без впливу на глобальну систему
- Уникати конфліктів версій між різними проектами
- Легко відтворювати середовище на інших машинах
- Управляти залежностями проекту

### Файл залежностей

Різні інструменти використовують різні формати файлів для опису залежностей:

- **requirements.txt** - традиційний формат для pip/venv
- **Pipfile** і **Pipfile.lock** - формат для Pipenv
- **pyproject.toml** - сучасний стандарт (PEP 518), використовується Poetry
